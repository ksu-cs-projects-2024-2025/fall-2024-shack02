GrooveMapper Data Model

Welcome to GrooveMapper! GrooveMapper is a simple web app for exploring Spotify music data through customizable graph database queries. Each link at the top of the page contains a different customizable graph query that can be used to find connections and insights into various Spotify music data. On this page, choose an artist to get an idea of the GrooveMapper graph data model. Hover over the nodes and connections on the graph to see data about each graph component.
